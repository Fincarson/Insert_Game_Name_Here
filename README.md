## 🛠️ How to Install
1. Install the IDE CLion
2. Install the library Allegro5 (Make sure it's installed on C: Drive)
3. Install the compiler LLVM MinGW
4. Install CMake
5. Set environment path for Allegro5, and LLVM MinGW
<br>
